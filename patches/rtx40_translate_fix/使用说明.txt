翻译功能修复补丁（RTX 50/40/30/20 通用）
========================================

适用场景：
  整合包翻译功能报错（日志含 0xc000001d / "translate failed" 500），
  或翻译模型加载即崩溃。本补丁同时支持 RTX 50 / 40 / 30 / 20 系显卡。


使用方法：
  1. 把以下两个文件放到整合包根目录
     （能看到 runtime 文件夹的那一层，通常是
       D:\AI-SD-ComfyUI\WD14-Tagger-Web-Portable\）：

       ① 修复翻译.bat
       ② llama_cpp_python-0.3.31-py3-none-win_amd64.whl

  2. 双击「修复翻译.bat」，等待出现「修复完成」字样。

  3. 关闭原来打开的整合包窗口（任务管理器确认没有 python.exe），
     重新双击整合包的启动脚本启动程序。

  4. 测试翻译功能，正常出结果即修复成功。


前置条件：
  NVIDIA 驱动版本 >= 580（在 cmd 里运行 nvidia-smi 可查看）。
  能正常跑 ComfyUI 的一般都满足。


原理（给好奇的人）：
  旧版整合包只打了 RTX 50（Blackwell）专属的翻译推理库，
  其他架构（RTX 40/30/20）缺少匹配的 GPU 内核，加载即崩。
  本补丁换成 0.3.31 版通用推理库——内置 sm_75/80/86/89/90 全架构
  GPU 内核 + sm_90 PTX，RTX 50/40/30/20 均原生或 JIT 适配，一个 wheel 通吃。


如果修复后仍崩溃：
  双击同目录的「诊断GPU加载.bat」，
  把生成的 diag_gpu_output.txt 发给作者。
